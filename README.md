## Description

This project is a example how create a graphql api using nest, here I use the most common features from nest to learn

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

http://localhost:3000/graphql
